HTML Cheatsheet: Tags

Tag Name Description
<div> Division Defines a division or section in an HTML document.
<p> Paragraph Represents a paragraph of text.
<a> Anchor Creates a hyperlink.
<img> Image Embeds an image in the document.
<ul> Unordered List Defines an unordered list.
<ol> Ordered List Defines an ordered list.
<li> List Item Represents a list item in a list.
<h1> to <h6> Headings (Heading 1 to Heading 6) Defines headings with different levels of importance.
<span> Span Inline element used for styling or scripting.
<input> Input Creates an input field for user input.
